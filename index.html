Distributed Consensus Visualisation
Through the creation of a framework in Python and Java, Distributed Consensus
Algorithms can now be easily visualized through Message Sequence Diagrams (MSDs).
This allows the user to gain a greater understanding of the algorithm, as well as faster
debugging times. This framework is planned to be used to teach over 20 second and
fourth year students at the University of Northern British Columbia to better understand
Distributed Consensus.
